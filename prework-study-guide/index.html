HTML

  • The head element contains information about the webpage.
  • The body element represents the visible contain shown to the user.
  • The break element is an element used to separate content by a line break.
  • The h1-h6 elements characterize the different headers on a webpage, usually in secuential order with 1 being the largest and 6 being the smallest.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A selector defines which elements or attributes to which rules apply
  • Declarations function as what styles we want the code to apply to our selectors
  • Property is the category of style that is applied to the elements/attriubutes
  • Value describes which selection is made in relation to its property to be applied to elements/attributes

GIT

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • cd: changes the directory we are currently in
  • code .: opens the current branch in VS code
